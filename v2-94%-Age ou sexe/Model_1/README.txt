

                                     ESTIMATION DE L'AGE A PARTIR DE LA VOIX


Pour lancer le modèle il faut : 
	- installer les librairies dans requirements.txt 
	- executer le fichier launch.py avec la commande : pyhton launch.py

Le dossier best_models represente le modèle déja entrainé avec une précision de 0.74 . Pour réentrainer le model, voir le notebook age_estimation.ipynb

Le dataset utilisé pour l'entrainement est Common Voice Corpus version française, traité avec le fichier traitement.py afin d'aplatir 
la gaussienne reprensenté par cette dataset (voir le fichier fig).
